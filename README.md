# Quasar App (scm-ui)

A Quasar Project

## Install the dependencies
```bash
npm install  
```

### to make build 
```bash
npm run build 
```

### to link 
```bash
npm link 
```

after link delete node_module from lib application  

### Customize the configuration
See [Configuring quasar.config.js](https://v2.quasar.dev/quasar-cli-webpack/quasar-config-js).
